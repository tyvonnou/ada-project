# ADA Project

The objective of this work is to develop software to simulate real-time scheduling algorithms.

## Compilation

* `gnat make -g -O0 -Ipackages exemple`

## Start

* `gdb exemple`
* `(gdb) run`

## Clean

### Windows

* `rm b* *.exe *.o *.ali`

### Linux

* `rm b* exemple *.o *.ali`
